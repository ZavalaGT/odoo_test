## Tech used

HTML5, CSS, Boostrap v4.6, Sass.

## Run code.

1. Clone repository

```bash
$ git clone https://github.com/ZavalaGT/odoo_test.git
```

2. Run

```bash
$ npm install
```

3. Install sass

```bash
$ npm install -g sass
```

4. For new custom values run

```bash
$ sass sass/custom.scss main.css
```

4. Open index.html file
